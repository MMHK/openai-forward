Welcome to OpenAI Forward API

This project provides a simple forward proxy for OpenAI's APIs. It allows you to easily access and use OpenAI's powerful language models in your applications.

Main Features:

  • Simple and easy-to-use API interface
  • Forward requests to OpenAI's servers securely
  • Supports all major OpenAI API endpoints

How to Use:

  1. Start the server by running the Go application
  2. Send your API requests to the local endpoint
  3. The proxy will handle the request and return the response from OpenAI
